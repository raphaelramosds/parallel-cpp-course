# Aula 18 - Redução em OpenMP

## Visão geral

A redução permite que múltiplas threads colaborem para calcular um valor agregado, como soma, produto, máximo ou mínimo, de forma eficiente e segura. Em OpenMP, a cláusula `reduction` é usada para especificar variáveis que serão reduzidas ao final da execução paralela.
